A linear data structure storing data whose top is maintained while entering/deleting. Consists mainly of following operations:
- `Push`: An element is added on top of previous element ,if exists. The element added becomes the new top.
- `Pop`: Top-most element is popped (deleted). The element below becomes the new top. If no data is present, returns null.
- `Top`: Returns the value of top-most element.
