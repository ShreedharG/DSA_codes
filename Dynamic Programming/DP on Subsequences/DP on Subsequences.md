Subsequences are continuous in nature of an array. For example arr = {1,2,3},
Its subsequence are : (1),(1,2),(1,3),(2),(2,3),(3). It is slightly different than subarray in the way that the elements in subsequence appear in same order as the main array but not necessary continuous in indices.
